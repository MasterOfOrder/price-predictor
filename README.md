# 🔮 AI-ზე დაფუძნებული ტექნიკის ფასების საპროგნოზო ასისტენტი

პროექტის მიზანია შექმნას ინტელექტუალური ასისტენტი, რომელიც მომხმარებლის მიერ მიწოდებული პროდუქტის ლინკის (URL) საფუძველზე ახდენს ფასების ისტორიის გენერირებას, აანალიზებს ბაზრის ტრენდებს მანქანური სწავლების (Machine Learning) მეშვეობით და მომხმარებელს აძლევს ზუსტ რეკომენდაციას: **ღირს თუ არა პროდუქტის ახლავე შეძენა, თუ სჯობს მოცდა.**

პროექტი აგებულია მოდულარული არქიტექტურით და აერთიანებს ნატურალური ენის დამუშავებას (NLP), დინამიურ პარსინგს, მონაცემთა ტრანსფორმაციასა და რეგრესიულ მოდელირებას რეალურ დროში (In-Memory).

---

## 🛠 პროექტის არქიტექტურა და მოდულები

პროექტი დაყოფილია რამდენიმე დამოუკიდებელ პაკეტად, რაც უზრუნველყოფს კოდის სისუფთავეს და მარტივ მასშტაბირებას:

### 1. `src/scraper/web_scraper.py` (ვალიდატორი და დინამიური ენჯინი)
* იღებს მომხმარებლის მიერ შეყვანილ ნებისმიერ ტექსტს და ამოწმებს, არის თუ არა ის ვალიდური URL ლინკი.
* არასწორი ლინკის ან უბრალო ტექსტის შემთხვევაში ბლოკავს მილსადენს (Pipeline) და აბრუნებს შეცდომას.
* ვალიდური ლინკიდან ექსტრაქციას უკეთებს დომენსა და პროდუქტის უნიკალურ იდენტიფიკატორს (Slug).
* კრიპტოგრაფიული ჰეშირებით (MD5-ზე დაფუძნებული Seed ენჯინით) ქმნის სტაბილურ, ინდივიდუალურ 90-დღიან ფასების ისტორიას ყოველი ახალი ლინკისთვის ყოველგვარი ხელით გაწერილი ქივორდების (Hardcoded keywords) გარეშე.

### 2. `src/predictor/data_loader.py` (მონაცემთა ტრანსფორმატორი)
* რეალურ დროში იჭერს სკრეიპერის მიერ RAM-ში გენერირებულ JSON მონაცემთა ნაკადს.
* გარდაქმნის ინფორმაციას Pandas DataFrame-ად და ახდენს თარიღების კონვერტაციას უწყვეტ ციფრულ Timeline ინდექსად (`Day_Index`).
* ამზადებს ორგანზომილებიან მასივებს ($X$ და $y$) Scikit-learn მოდელისთვის.

### 3. `src/predictor/train.py` (მანქანური სწავლება და პროგნოზი)
* იყენებს **ხაზოვან რეგრესიას (Linear Regression)** მომავალი 7 დღის ფასების ექსტრაპოლაციისთვის. 
* ხაზოვანი მოდელი შერჩეულ იქნა სპეციალურად, რათა თავიდან აგვეცილებინა ხეზე დაფუძნებული მოდელების (მაგ. Random Forest, Gradient Boosting) კლასიკური პრობლემა – ე.წ. "Tree Extrapolation Problem" (როდესაც მოდელი ტრენინგ მონაცემების მაქსიმალურ ზღვარს სცდება და მომავალზე ერთსა და იმავე ციფრს აბრუნებს).
* მომავალ ფასებს უმატებს მცირე საბაზრო მიკრო-ვარიაციებს (Gaussian Noise) რეალისტური გრაფიკების მისაღებად.

### 4. `src/bot_logic/decision_engine.py` (AI ასისტენტის ლოგიკა)
* იყენებს მოწინავე NLP ტექნოლოგიას **Zero-Shot Classification**-ს (`facebook/bart-large-mnli` მოდელს Hugging Face Transformers-იდან).
* წინასწარი ტრენინგის გარე
